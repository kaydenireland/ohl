# Ohl Language Support

VS Code extension for the Ohl programming language

This extension provides syntax highlighting, and execution of ```.ohl``` files directly from VS Code.

-------

## Features

### Language Support

- Syntax highlighting
- Direct execution

-------

## Requirements

The Ohl Cli (oo) must be installed and available on your system ```PATH```

To verify:

```bash
oo --version
```

If the command is not found, install and add oo to your ```PATH``` before running files.
